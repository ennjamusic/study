В /engine/proected/config/config.php находятся константы, изменяя которые можно менять шаблон и язык приложения(DEFAULT_LANG,TEMPLATE_PATH). Также, там можно поменять переменные для подключения к базе данных. Приложение написано для базы MySQL, но, поменяв в файле конфигурации контанту DSN, можно запустить и на других базах, которые поддерживает PDO.
